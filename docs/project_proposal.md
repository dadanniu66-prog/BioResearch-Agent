# 项目申报书：BioResearch-Agent

## 1. 项目名称

BioResearch-Agent：面向医学本科生的 AI 驱动生物医学科研工作流智能体

## 2. 项目背景

当前医学科研正在快速进入数据驱动阶段。GEO、NHANES、单细胞转录组数据库等公共资源中包含大量可用于疾病机制研究、标志物筛选和临床风险预测的数据。

但是，对于医学本科生和科研初学者来说，想要独立完成一个完整的生物信息学项目并不容易。一个完整项目往往需要同时掌握公共数据库检索、R 语言编程、表达矩阵整理、样本分组、差异分析、WGCNA、机器学习、富集分析、单细胞分析以及科研写作等多方面能力。

BioResearch-Agent 希望解决这个问题。它的目标不是简单让 AI 写几段代码，而是构建一个能够参与科研流程设计、代码生成、错误诊断、结果解释和报告生成的多 Agent 协作系统。

## 3. 核心痛点

本项目主要解决以下问题：

1. 医学本科生很难判断一个公共数据集是否适合自己的研究方向。
2. GEO、NHANES 和单细胞数据的文件结构复杂，初学者容易卡在数据读取和整理阶段。
3. R 语言代码运行中经常出现路径错误、包安装错误、矩阵格式错误等问题。
4. 很多学生即使跑出了图，也不知道每张图在科研逻辑中说明什么。
5. 生物信息学分析流程容易碎片化，缺乏可复现的项目结构。
6. 从分析结果过渡到大创申报书、开题报告或论文初稿时，缺少系统化辅助。

## 4. 项目目标

BioResearch-Agent 的目标是构建一个面向医学科研训练的 AI Agent 系统，帮助用户完成从科研想法到数据分析，再到结果解释和报告生成的完整流程。

项目计划支持以下任务：

- 公共数据库理解
- GEO 表达矩阵整理
- 样本分组构建
- 差异表达分析
- WGCNA 共表达网络分析
- 机器学习筛选关键基因
- GO / KEGG / GSEA 富集分析
- 单细胞数据验证
- NHANES 队列分析
- 科研图表解释
- 大创申报书和论文框架生成

## 5. 核心逻辑

BioResearch-Agent 采用多 Agent 协作模式。

系统主要包括以下 Agent：

### 5.1 数据库理解 Agent

负责读取 GEO、NHANES 或单细胞数据库的说明信息，判断数据类型、样本分组、平台信息、可用文件和适合的分析策略。

### 5.2 R 代码生成 Agent

负责根据具体任务生成 R 脚本，例如表达矩阵整理、差异分析、WGCNA、机器学习、富集分析和可视化代码。

### 5.3 错误诊断 Agent

负责读取 R 语言运行日志、终端输出和报错信息，判断代码是否成功运行，并给出修改建议。

### 5.4 结果解释 Agent

负责把生物信息学结果转化成医学科研语言，解释每张图说明了什么、与疾病机制有什么关系、如何组织成科研故事线。

### 5.5 科研报告 Agent

负责把分析结果整理成大创申报书、课题设计方案、论文初稿框架、图注说明和汇报材料。

## 6. 示例应用场景

本项目可以应用于以下医学科研方向：

- 非酒精性脂肪肝相关机制研究
- 糖尿病肾病关键基因筛选
- 糖尿病足单细胞分析
- 蛋白修饰相关基因集研究
- 线粒体代谢和酮体代谢相关基因研究
- NHANES 癌症幸存者队列分析
- bulk RNA-seq 与单细胞数据联合验证

## 7. 当前进展

目前项目处于原型设计和工作流验证阶段，已经初步完成以下内容：

- 医学科研 Agent 系统的总体设计
- GEO 数据集处理流程设计
- R 脚本自动生成思路设计
- 表达矩阵整理和样本分组流程设计
- R 运行报错诊断流程设计
- 生信结果图表解释框架设计
- 大创申报书和科研文书生成流程设计

## 8. 预期成果

项目预期形成以下成果：

1. 一个标准化的 AI 辅助生物医学数据分析工作流。
2. 一组可复用的 R 语言分析脚本模板。
3. 一组面向不同任务的 Agent Prompt 模板。
4. 一个适合医学本科生学习的中文科研智能体原型。
5. 一个能够把公共数据库分析结果转化为科研申报材料和论文框架的工具雏形。

## 9. 项目创新点

本项目的创新点在于将 AI Agent 工作流与医学科研训练结合起来。

传统 AI 辅助科研往往只是让模型回答问题或生成单段代码，而 BioResearch-Agent 更强调完整流程：

```text
科研问题提出
    ↓
公共数据库理解
    ↓
数据整理
    ↓
R 代码生成
    ↓
错误诊断
    ↓
结果解释
    ↓
科研报告生成
这种方式可以帮助医学本科生更快理解生物信息学科研的完整逻辑，降低入门门槛，提高项目复现性。

11. 总结

BioResearch-Agent 是一个面向医学本科生和科研初学者的 AI 驱动科研工作流项目。它希望通过多 Agent 协作，把复杂的生物医学数据分析流程拆解成可学习、可执行、可复现的步骤。

本项目需要大量 token 支持，因为它涉及长文档阅读、R 代码生成、多轮调试、科研逻辑解释和报告生成等高 token 消耗任务。获得 token 支持后，项目将进一步完善为一个可用于医学科研训练和公共数据库分析的 AI Agent 原型。
